# TU_listing_nodeJS_TD
<h2>Listing with unit tests</h2>
TD de NodeJS avec l'utilisation de tests unitaires. <br />
Lancer le projet : <strong>npm install</strong> <br />
Lancer les tests : <strong>npm run test:watch</strong> <br />

<h2>Liste des scenarios :</h2>
<ul>
<li>Scenario 1 : Créer une liste avec un nom</li>
<li>Scenario 2: Supprimer une liste avec un nom</li>
<li>Scenario 3 : Afficher les listes de courses</li>
<li>Scenario 4: Ajouter un nouvel élément à partir d'un nom à une liste de course donnée</li>
<li>Scenario 5: Lister les articles d'une liste de course</li>
<li>Scenario 6: Flaguer des articles comme OK (acheté)</li>
<li>Scenario 7 (AJOUTÉ) : Supprimer un article si l'on s'est trompé</li>
<li>Scenario 0 : Article / Liste : UUID v4 généré aléatoirement</li>
</ul>
